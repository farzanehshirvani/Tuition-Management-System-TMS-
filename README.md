# Tuition-Management-System-(TMS)
شرح سیستم

عنوان| توضیحات
نام سیستم| Tuition Management System for Educational Complex
هدف سیستم| مدیریت اطلاعات آموزشی و مالی دانش‌آموزان، ثبت شهریه، کنترل پرداخت‌ها و ارائه گزارش‌های مالی در مجتمع آموزشی.
کاربران سیستم| مدیر مجتمع آموزشی، مسئول امور مالی، مسئول ثبت‌نام و دانش‌آموز (برای مشاهده وضعیت مالی).
زیرسیستم مدیریت دانش‌آموز| ثبت، ویرایش، حذف و جستجوی اطلاعات دانش‌آموزان و تخصیص آن‌ها به مدرسه، پایه و کلاس.
زیرسیستم مدیریت مدارس، پایه و کلاس| ایجاد و مدیریت مدارس، پایه‌های تحصیلی، کلاس‌ها و تعیین مبلغ شهریه هر کلاس.
زیرسیستم پرداخت شهریه| ثبت پرداخت‌های نقدی، کارت بانکی و اقساطی، ثبت تاریخ پرداخت و مبلغ هر تراکنش و امکان مدیریت اقساط.
زیرسیستم پرونده مالی| ایجاد پرونده مالی برای هر دانش‌آموز، محاسبه مجموع پرداخت‌ها، مانده بدهی، وضعیت تسویه و نگهداری سوابق مالی.
گزارش‌ها| نمایش وضعیت مالی هر دانش‌آموز، میزان شهریه، مجموع پرداخت‌ها، مانده بدهی و گزارش دانش‌آموزان بدهکار یا تسویه‌شده.
مزایای سیستم| کاهش خطاهای مالی، افزایش سرعت ثبت اطلاعات، مدیریت دقیق پرداخت‌ها، امکان پیگیری اقساط و دسترسی سریع به گزارش‌های مالی.

جمع‌بندی

سیستم مدیریت شهریه مجتمع آموزشی با یکپارچه‌سازی اطلاعات آموزشی و مالی، فرآیند ثبت شهریه، پرداخت، مدیریت اقساط و نگهداری پرونده مالی دانش‌آموزان را ساده و دقیق می‌کند. این سامانه باعث کاهش خطاهای انسانی، افزایش سرعت انجام امور مالی و ارائه گزارش‌های دقیق برای تصمیم‌گیری مدیران مجموعه آموزشی می‌شود.
