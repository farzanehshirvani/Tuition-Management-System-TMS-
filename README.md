# Tuition-Management-System-(TMS)
# سیستم مدیریت شهریه مجتمع آموزشی

## شرح سیستم

| بخش | توضیحات |
|------|---------|
| **نام پروژه** | سیستم مدیریت شهریه مجتمع آموزشی |
| **هدف سیستم** | مدیریت اطلاعات دانش‌آموزان، شهریه، پرداخت‌ها و پرونده مالی در یک مجتمع آموزشی شامل چندین مدرسه. |
| **کاربران سیستم** | مدیر مجتمع، مسئول مالی، مسئول ثبت‌نام و دانش‌آموز |
| **مدیریت دانش‌آموز** | ثبت، ویرایش، حذف و جستجوی اطلاعات دانش‌آموزان و تخصیص آن‌ها به مدرسه، پایه و کلاس. |
| **مدیریت مدارس و کلاس‌ها** | تعریف مدارس، پایه‌های تحصیلی، کلاس‌ها و تعیین مبلغ شهریه هر کلاس. |
| **مدیریت پرداخت** | ثبت انواع پرداخت شامل نقدی، کارت بانکی و پرداخت اقساطی همراه با ثبت تاریخ و مبلغ هر پرداخت. |
| **مدیریت پرونده مالی** | ایجاد پرونده مالی برای هر دانش‌آموز، محاسبه مبلغ پرداخت‌شده، مانده بدهی، وضعیت اقساط و سوابق مالی. |
| **گزارش‌ها** | مشاهده وضعیت مالی دانش‌آموزان، میزان بدهی، پرداخت‌ها، اقساط و گزارش دانش‌آموزان تسویه‌شده یا بدهکار. |
| **مزایای سیستم** | کاهش خطاهای مالی، افزایش سرعت مدیریت شهریه، امکان پیگیری اقساط، دسترسی سریع به اطلاعات و ارائه گزارش‌های دقیق. |

## خلاصه

سیستم مدیریت شهریه مجتمع آموزشی با یکپارچه‌سازی اطلاعات آموزشی و مالی، فرآیند ثبت شهریه، پرداخت‌ها، مدیریت اقساط و نگهداری پرونده مالی دانش‌آموزان را ساده و دقیق می‌کند. این سامانه باعث افزایش دقت، کاهش خطاهای انسانی و تسهیل مدیریت امور مالی در مدارس یک مجتمع آموزشی می‌شود.
